# Text-on-screen

A simple web application that allows you to add and position text on top of a background image.

## Features

- **Click to add text**: Click anywhere on the background to create a new text box
- **Drag to move**: Drag text boxes to reposition them anywhere on the screen
- **Print Page**: Opens the print dialog to print the background with all text overlays
- **Reset Page**: Clears all text boxes from the screen

## Usage

1. Open `index.html` in a web browser
2. Click anywhere on the background to create a text box
3. Type your desired text
4. Drag text boxes to reposition them
5. Use "Print Page" to print your creation
6. Use "Reset Page" to start over

## Files

- `index.html` - Main HTML structure
- `styles.css` - Styling for the application
- `script.js` - JavaScript functionality for text box creation and manipulation
- `background.png` - Background image (replace with your own PNG)
